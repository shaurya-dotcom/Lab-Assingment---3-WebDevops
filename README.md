This is a one-page responsive personal portfolio website created for Web Dev Lab Assignment 3.

## Features

- Sections: Header, Hero, About, Skills, Projects, Contact, Footer
- Flexbox used for the header and hero layout
- CSS Grid used for the Skills and Projects cards
- Responsive design with media queries for tablet and mobile screens
- Relative CSS units (rem, %, vw, vh) for spacing and sizing
- Hover effects and CSS transitions on navigation links, cards and button
- Simple keyframe animation on the hero title text
- Contact form with name, email and message fields
